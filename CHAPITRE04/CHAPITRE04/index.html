1)Introduction:

1-Internet :support de communication général.

2-Web: une partie des contenus circulant sur l’Internet. Le Web permet de consulter, avec un navigateur, des pages accessibles sur des sites.

3-Principe World Wide Web :

-Accéder à des documents textuels.

-Situés sur des machines accessibles.

-Actuellement : servir des ressources.

-De différentes natures : texte, image, son, vidéo…

4-URL (pour Uniform Resource Locator):C’est le nom unique de la page. décrivant l’emplacement d’une ressource. elle contient : L’adresse du serveur et l’adresse de la page sur le serveur.

2) Aspects techniques du Web :

1-URL :

a.Le Web permet d’accéder à un ensemble de ressources.

b.Le mécanisme de localisation peut faire appel au protocole DNS

2-HTTP qui est un Protocole de niveau applicatif , Paradigme client-serveur, Protocole sans état

3-HTML :

a.Langage de description de « pages Web »

b.Texte, images et autres objets

c.Programmation déclarative

3) Page web et site web :

Une page Web, c’est un document multimédia:

• Avec un langage de description :HTML, XHTML ou HTML5.

•Stocké dans un fichier : html (ou .htm, .xhtml).

•Une page Web est interprétée et affichée par un navigateur.

•Un site web est un regroupement de pages Web.

•Emanant d’une même entité ,accessibles à partir d’une page d’accueil commune.

4)Création d’une page web HTML :

Un document HTML est :

Un fichier texte, construit suivant une syntaxe précise, Structuré par des balises et Interprété dans un navigateur ,Créé dans un éditeur de pages web ou tout éditeur de texte .

Les principes de langage HTML et Les balises:

Un fichier HTML doit comporter au minimum ces 4 balises :

A/ 1-html> … /html>: Délimite le début et la fin du document.

2-head> … /head> : Entête du document.

3-title> … /title> : Titre du document .

4-body> … /body> : Corps du document.

B/Les titres et paragraphes :

1-hn> … hn> : Titre de niveau n :1 à 6.

2-p> … p> : paragraphe.
