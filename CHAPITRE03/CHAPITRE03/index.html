1) Les Composants matériel d’un Réseau informatique

• HUB :

est un composant matériel utilisé pour connecter plusieurs appareils sur un réseau local (LAN). Il reçoit les données envoyées par un appareil et les transmet à tous les autres appareils connectés.

• Les cartes réseau (NIC):

C'EST des composants matériels qui permettent à un ordinateur de se connecter à un réseau. Elles permettent à l'ordinateur de transmettre et de recevoir des donner,Elles sont installées à l'intérieur de l'ordinateur

• SWITCH :

est un composant matériel utilisé pour connecter des appareils sur un réseau local (LAN) et qui permet à plusieurs appareils de communiquer entre eux en utilisant des adresses MAC (Media Access Control).

2) Le Modèle OSI

Le modele OSI (Open Systems Interconnection) est un modèle de référence pour les communications de réseau.

Les sept couches du modèle OSI sont les suivantes:

1• La couche physique : cette couche est responsable de la transmission des données sur le support physique.

2• La couche liaison de données :cette couche est responsable de la transmission des données entre les appareils qui partagent un même support physique.

3• La couche réseau : cette couche est responsable du routage des données à travers le réseau.

4• La couche transport :cette couche est responsable de l'acheminement des données de bout en bout entre les Differentes appareils.

5• La couche session : cette couche est responsable de l'établissement et de la gestion des sessions entre les appareils.

6• La couche présentation :cette couche est responsable de la conversion des données dans un format qui peut être compris par l'appareil de destination.

7• La couche application : : cette couche est responsable de la communication entre les applications sur les appareils de réseau.

3) Le Modèle TCP/IP

TCP/IP (Transmission Control Protocol/Internet Protocol) c'est un modèle de référence pour les communications de réseau, qui est utilisé pour la conception et la mise en œuvre des réseaux informatiques modernes.

Les quatre couches principales de modèle TCP/IP :

1•La couche liaison de données :: cette couche est responsable de la transmission des données sur le support physique

2•La couche réseau : : cette couche est responsable du routage des données à travers le réseau.

3•La couche transport : cette couche est responsable de l'acheminement des données de bout en bout entre les appareils.

4•La couche application : cette couche est responsable de la communication entre les applications sur les appareils de réseau.

4) L’adressage

un adressage IP: Une adresse IP est le numéro qui identifie chaque ordinateur dans les réseaux qu’il appartient. Il y’a deux mode d’adressage IPV4 (Chaque paquet contient : adresse source et une adresse destinataire sur (32 bits)) et IPV6 (Les adresses sont codées sur 128 bits)

5)L’adresse IPV4

-Les différents types d’adresse IPV4:

1) L’adresse réseau : tous les bits de la partie hôtes=0, c'est-à-dire tous les hôtes de ce réseau ont la même partie réseau

2) L’adresse de diffusion : il s’agit d’une adresse dans laquelle tous les bits de partie hôtes=1, et tous les hôtes de cette adresse reçoivent ces données, sans qu’il distribue un paquet pour chaque pc d’une manière unique.

3) L’adresse hôtes : : On attribue les valeurs situés entre l’adresse réseau et l’adresse de diffusion sur machines situé sur cette réseau.

4) Le masque de réseau et de sous réseaux : Tous les bits de la partie réseaux = 1
